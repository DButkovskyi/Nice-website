Your own website: prerequisites

00
Who is it for?

I am creating this guide for people who want to start learning web development and have some fun. Our final goal is to create a simple STATIC website, so if you already have experience creating websites I would recommend you to skip this guide. You also will not find tons of text about how to code, instead I will provide you with a simple step by step guide and some useful links if you need to learn more.
This guide will walk you through all the instructions for technologies that will be needed for developing and deployment, so completion will provide you with a perfect knowledge base for your next, more advanced projects.So, what will we use?

01
Code Editor

First of all we will need a code editor.
I personally like Visual Studio Code and would recommend it for couple reasons:

  • It's free
  • Easy to setup
  • Customisation: there are many extensions you can install

You also can search and try different options to find a perfect fit for you.

02
HTML/CSS

HTML/CSS are the main tools we will be using for development, so I would recommend start by learning basics, freecodecamp might be the best place to start, just remember: you don't have to complete the whole course, as soon as you understand HTML tags, paddings/margins and CSS selectors you can start working on the website, doing is the best way to learn. I will help you to create a structure of the website and provide you with required instructions and helpful links.

03
Deploying

After stage two you should have your files ready for launch and for our last step we will be using Amazon Web Services, (AWS) to launch it, so you will need an AWS account(it's free), the only paid thing will be domain name.
